### Auto Lead Gen with Airtable and Zapier
**Tool:** Zapier
**Desc:** Generate leads from a Notion database and store them in Airtable for follow-up
**Prompt:** Create a Zap that captures new entries in a Notion database and stores them in an Airtable base. Use the blueprint: {revenue_target}, {seller_role}. Set the Airtable base to 'leads' and the Notion database to '{{ $json.state.niche }}'.
**Hook:** Lead generation and sales follow-up
